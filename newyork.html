This capstone project is designed for the Spring 2025 showcase. It presents an interactive data visualization that merges weather data (temperature, rain, snow, and wind) with Google search term trends (results and patterns). The goal is to deliver an engaging and interactive experience for users.



This demo is developed using HTML, CSS, JavaScript, and Python, utilizing the D3.js framework for data visualization. It also incorporates the Google Custom Search JSON API and Google Trends API for search data, as well as NOAA's weather data in JSON format.
