IN THIS EDITION:

This capstone project, developed for the Spring 2025 showcase, features an interactive data visualization that combines weather data (temperature, rainfall, snowfall, and wind) with trends in Google search terms. The objective is to provide users with an engaging and insightful interactive experience.



This demo is built using HTML, CSS, JavaScript, and Python, with data visualization powered by the D3.js framework. It integrates search data via the Google Custom Search JSON API and Google Trends API, and utilizes weather data provided in JSON format by NOAA.

INSTRUCTIONS TO INTERACT WITH DATA:

Below is an interactive data visualization with buttons to explore different search terms. Clicking a button transitions the graph to new data, adjusts the axes, and plays a sound. The temperature is shown as a blue line graph that plays automatically from the start and can be reset using the controls.

Interact with the Temperature: RESET TEMPERATATURE SOUND = "R" key. PLAY AFTER RESET = "Space" key.
